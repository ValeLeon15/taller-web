DESCRIPCIONES DE ENDPOINTS

Endpoints del Controlador ArrendadorController:

  • Obtener un arrendador por ID: Método: GET, URL: /progrupo14/arrendadores/{id}, Produces: JSON, Descripción: Este endpoint permite obtener un arrendador específico según su ID.
  • Obtener todos los arrendadores: Método: GET, URL: /progrupo14/arrendadores, Produces: JSON, Descripción: Este endpoint devuelve una lista de todos los arrendadores disponibles.
  • Guardar un nuevo arrendador: Método: POST, URL: /progrupo14/arrendadores, Produces: JSON, Descripción: Este endpoint permite guardar un nuevo arrendador. Espera recibir un objeto JSON en el cuerpo de la solicitud que representa al nuevo arrendador.
  • Actualizar un arrendador existente: Método: PUT, URL: /progrupo14/arrendadores, Produces: JSON, Descripción: Este endpoint permite actualizar la información de un arrendador existente. Espera recibir un objeto JSON en el cuerpo de la solicitud que contiene los nuevos datos del arrendador a actualizar.
  • Eliminar un arrendador por ID: Método: DELETE, URL: /progrupo14/arrendadores/{id}, Produces: JSON, Descripción: Este endpoint permite eliminar un arrendador específico según su ID.
